# MinePulse

MinePulse is a futuristic Minecraft server intelligence platform built using HTML, CSS, and JavaScript.

## Features

- Real-time Minecraft server status
- Multi-server monitoring
- Favorite servers system
- Search history
- Bedrock detection
- Server software detection
- Responsive futuristic UI
- Animated particle background
- Local storage analytics
- Live dashboard interface

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- mcsrvstat.us API
- Particles.js
